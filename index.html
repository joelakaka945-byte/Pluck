error.api.invalid_body - Invalid payload

Cause: filenameStyle: 'custom' or empty enum. Fix: this build validates - allowed styles classic / pretty / basic / nerdy, modes auto / audio / mute, qualities max / 1080 / 720 / 480, audioFormats best / mp3 / ogg / wav / opus. Custom names are sanitized locally and sent as classic.

error.api.link.unsupported - Link Not Supported

Use a direct public https:// link that the engine supports. Private / login-only links fail by design.
