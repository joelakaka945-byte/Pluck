Error Guide read this before you stress

If Pluck does not work, it is almost always one of these reasons. Here is what they actually mean, in plain bro plus pro language.

Link Not Supported - Wrong website bro

Professional reason: The domain you pasted is not in the supported extractor list. Bro translation: Pluck works with major platforms like YouTube, TikTok, Instagram, X, Reddit. It does not work with Google Drive, Spotify, Netflix, or private sites. Solution: Make sure your link is a public video post from a supported app.

Private or Unavailable - Video is locked

Professional reason: Content is private, deleted, members only, or region blocked. Bro translation: If you cannot open the link in incognito without logging in, Pluck cannot either. We do not bypass logins. Solution: Use a public link or make sure the video is still live.

Age Restricted - 18 plus wall

Professional reason: The platform requires age verification via logged in account. Cobalt intentionally does not bypass age gates. Bro translation: YouTube put an age lock on it. Solution: Confirm you can watch it while logged in on YouTube. If it needs login, it cannot be downloaded here.

Bad URL / Paste a link first - Input error

Professional reason: The input is empty or not a valid URL format. Bro translation: You pasted nothing or just text like “funny video”. Solution: Copy the full link using the Share button. It must start with https://

Engine Warming Up - Server was sleeping

Professional reason: The Railway host sleeps after inactivity. First request takes longer. Bro translation: The engine takes a nap to save money. Solution: Click Wake Engine and wait 8 to 10 seconds, then try again. The dot will turn green when ready.

No File Returned - Empty response

Professional reason: The API returned 200 but no downloadable URL. Usually means livestream, DRM protected content, or expired story. Bro translation: The engine answered but had nothing to give. Solution: Try 720p instead of Maximum, or check if the video is a live or premium protected video.

Request Failed / Timeout - Network hiccup

Professional reason: Fetch timed out after 20 seconds or returned HTTP error. Bro translation: Your WiFi hiccuped or Railway was slow. This is the bug that refresh fixes. Solution: Hit Refresh Page to Fix Bugs above. This wipes stacked state and gives you a clean retry. Works 99 percent of the time.

Engine must respond at https://cobalt-production-2c27.up.railway.app - if not, click Wake Engine
