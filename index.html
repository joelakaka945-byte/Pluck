Answer: Pluck is a clean user interface. You paste a link, we create a JSON object containing url, videoQuality, filenameStyle, disableMetadata, downloadMode auto, and audioFormat best, and send a POST request to your base URL. The engine then locates the direct MP4 file. We only process the returned url and filename to display the preview, trim editor, and download options.
Trim without typing numbers: After the video loads, a highlighted Trim Editor appears above the download buttons. Drag the left handle to set the start and the right handle to set the end. Use Preview Selection to watch only the selected part. Click Confirm Trim, then click Download Trimmed Video. You can still download the original untrimmed video using the button below. The trim is processed in your browser.
  • 1 Paste: Copy the full link using the Share button. The link must start with https://
  • 2 Click Download: The main tool is at the top of the page.
  • 3 Wait for ready: The file appears with a Trim Editor highlighted at the top.
  • 4 Drag to trim: Move the two handles on the timeline. No numbers needed.
  • 5 Preview and Confirm: Click Preview Selection, then Confirm Trim.
  • 6 Download: Click Download Trimmed Video, or Download Original Untrimmed Video to keep the full file.
  • Download Trimmed Video: Highlighted primary button inside the Trim Editor, appears after you confirm. Downloads only the selected portion.
  • Download Original Untrimmed Video: Keeps the full length, same as the normal direct download.
  • Direct Download: Best for Android and Desktop, saves to Downloads folder.
  • Blob Save: Optimized for iPhone where the download attribute is ignored.
  • File Picker Save: Opens a system dialog to choose a folder on modern desktop browsers.

Error Guide Read this before you worry

Quick reference, also explained in the FAQ above.

Link Not Supported - Incorrect website

Please use a public link from Instagram, TikTok, YouTube, X, or Reddit.

Private or Unavailable - Video is locked

If it cannot be viewed in incognito mode, the engine cannot access it either.

Age Restricted - Age verification required

Login is required, the engine does not bypass restrictions.

Bad URL - Input error

The link must start with https:// and be copied from the Share button.

Engine Warming Up - Engine is in sleep mode

Click Wake Engine and wait 8 to 10 seconds.

No File Returned - No content found

This may be a livestream, DRM protected content, or an expired file. Try 720p.

Timeout - Network interruption

Click Refresh Page to Fix Issues to clear all stored data.

Engine must respond at https://cobalt-production-2c27.up.railway.app - If not responding, click Wake Engine
