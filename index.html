Error Guide for Twins read this before you stress

If Pluck does not work, it is almost always one of these. Here is what they mean in plain twin plus pro language, with highlights on what to do.

Link Not Supported - Wrong website twin

Highlight: Works on Instagram, TikTok, YouTube, X, Reddit. Pro reason: domain not in supported list. Fix: use a direct public post link from those apps.

Private or Unavailable - Video is locked twin

Highlight: If incognito cannot see it, Pluck cannot either. Pro reason: private, deleted, members only, or region blocked. Make sure the Instagram or TikTok is public.

Age Restricted - 18 plus wall twin

Pro reason: needs age verification. Cobalt does not bypass age gates by design. Open it logged in to confirm.

Bad URL - Input error twin

Pro reason: input empty or not URL. Fix: copy full link starting with https:// from Share button on Instagram or TikTok.

Engine Warming Up - Server sleeping twin

Highlight: Free hosts sleep. Click Wake Engine and wait 8 to 10 seconds, dot turns green when ready.

No File Returned - Empty response twin

Pro reason: API gave no URL. Usually livestream, DRM, or expired story. Try 720p instead of Maximum.

Timeout - Network hiccup twin

Highlight: Most common fix. Hit Refresh Page to Fix Bugs, it wipes stacked state from A to Z and works 99 percent of time.
