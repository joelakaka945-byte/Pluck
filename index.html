Error Guide read this before you stress

If Pluck does not work, it is almost always one of these. Here is what they mean in plain bro plus pro language.

Link Not Supported - Wrong website bro

Pro reason: domain not in supported extractor list. Bro translation: Pluck works with YouTube, TikTok, Instagram, X, Reddit. Not Drive, Spotify, Netflix. Use a public video link.

Private or Unavailable - Video is locked

Pro reason: content is private, deleted, members only, or region blocked. Bro translation: If incognito cannot see it without login, Pluck cannot either.

Age Restricted - 18 plus wall

Pro reason: platform needs age verification. Bro translation: YouTube put an age lock. Cobalt does not bypass it by design.

Bad URL - Input error

Pro reason: input empty or not a URL. Bro translation: you pasted nothing or just text. Solution: copy full link starting with https://

Engine Warming Up - Server sleeping

Pro reason: host sleeps after inactivity. Bro translation: engine naps to save money. Click Wake Engine and wait 8 to 10 seconds.

No File Returned - Empty response

Pro reason: API gave no URL. Usually livestream, DRM, or expired story. Try 720p instead of Maximum.

Timeout - Network hiccup

Pro reason: fetch timed out. Bro translation: WiFi or Railway lagged. Hit Refresh Page to Fix Bugs, works 99 percent of time.

Engine must respond at https://cobalt-production-2c27.up.railway.app - if not, click Wake Engine
